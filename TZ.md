# Техническое задание на проект: Freelance Cashflow Predictor

## 1. Цель проекта

Разработка веб-приложения для учета доходов фрилансера.  
Проект позволяет хранить данные об инвойсах, импортировать их из CSV-файлов, анализировать историю доходов и получать простой прогноз месячного cashflow на основе предыдущих данных.

## 2. Роли пользователей

* **Гость** — может зарегистрироваться и войти в систему.
* **Зарегистрированный пользователь** — ведет учет своих инвойсов, импортирует данные из CSV, просматривает список инвойсов и дашборд с графиком доходов.
* **Администратор** — управляет всеми данными через стандартную панель Django Admin.

## 3. Модели данных

В проекте используются следующие пользовательские модели данных:

### Client
* owner (ForeignKey → User)
* name
* email
* notes

### Project
* owner (ForeignKey → User)
* client (ForeignKey → Client)
* title
* description
* hourly_rate

### Category
* owner (ForeignKey → User)
* name

### Invoice
* owner (ForeignKey → User)
* project (ForeignKey → Project)
* category (ForeignKey → Category)
* date
* amount
* paid
* description
* external_id

## 4. Ключевой функционал

* Регистрация и авторизация пользователей.
* Добавление и редактирование инвойсов через веб-формы.
* Импорт инвойсов из CSV-файлов с проверкой данных.
* Просмотр списка инвойсов с пагинацией.
* Отображение графика доходов по месяцам.
* Прогноз будущего дохода на основе исторических данных.
* Экспорт данных в CSV-файл.

## 5. Используемые библиотеки и аналитика

* Работа с данными — **Pandas**.
* Прогнозирование — **scikit-learn (LinearRegression)**.
* Визуализация данных — **Plotly**.

## 6. Соответствие требованиям

* В проекте реализовано более трех пользовательских моделей данных.
* Используется база данных и Django ORM для работы с данными.
* Реализованы формы с серверной валидацией.
* Выполнена интеграция аналитических библиотек и визуализация данных.
* Проект реализован с использованием фреймворка Django.
