# Техническое задание на проект: Freelance Cashflow Predictor

## 1. Цель проекта

Создание простого веб-сервиса для учета доходов фрилансера. Проект позволяет добавлять и хранить инвойсы, импортировать данные из CSV-файлов, просматривать историю доходов и получать простой прогноз будущего месячного дохода.

## 2. Роли пользователей

* **Гость** — может зарегистрироваться и войти в систему.
* **Зарегистрированный пользователь** — управляет своими инвойсами, загружает CSV-файлы, просматривает список инвойсов и дашборд с графиком доходов.
* **Администратор** — управляет всеми данными через Django Admin.

## 3. Модели данных

В проекте используются следующие пользовательские модели:

* **Client**

  * owner (ForeignKey → User)
  * name
  * email
  * notes

* **Project**

  * owner (ForeignKey → User)
  * client (ForeignKey → Client)
  * title
  * description
  * hourly_rate

* **Category**

  * owner (ForeignKey → User)
  * name

* **Invoice**

  * owner (ForeignKey → User)
  * project (ForeignKey → Project)
  * category (ForeignKey → Category)
  * date
  * amount
  * paid
  * description
  * external_id

## 4. Ключевой функционал

* Регистрация и авторизация пользователей.
* Добавление инвойсов через веб-форму.
* Импорт инвойсов из CSV-файла.
* Просмотр списка инвойсов с пагинацией.
* Отображение графика доходов по месяцам.
* Прогноз будущего дохода на основе исторических данных.
* Экспорт данных в CSV (при необходимости).

## 5. Аналитика и библиотеки

* Анализ данных: **Pandas**.
* Прогнозирование: **scikit-learn (LinearRegression)**.
* Визуализация данных: **Plotly**.

## 6. Соответствие требованиям

* Используется более 3 пользовательских моделей.
* Реализована работа с базой данных через Django ORM.
* Присутствуют формы с серверной валидацией.
* Используется аналитическая библиотека Pandas и визуализация графиков.
* Проект реализован на Django и готов к деплою.
