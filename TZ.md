# Техническое задание на проект: Freelance Cashflow Predictor

1. Цель проекта
Проект предназначен для индивидуальных фрилансеров и мелких агентств. Он позволяет собирать данные о поступлениях (инвойсах), анализировать историческую динамику доходов и строить прогноз месячного cashflow на основе прошлых данных.

2. Роли пользователей
- Гость: может зарегистрироваться и просмотреть общую страницу с описанием.
- Зарегистрированный пользователь: вести личную базу инвойсов, импортировать CSV, просматривать дашборд и экспортировать данные.
- Администратор: управлять всеми моделями через Django admin.

3. Модели данных (минимум 3)
- Client: owner (User FK), name, email, notes.
- Project: owner (User FK), client (FK), title, description, hourly_rate.
- Category: owner (User FK), name.
- Invoice: owner (User FK), project (FK), category (FK), date, amount, paid, description, external_id.

4. Ключевой функционал (User Stories)
- Пользователь загружает CSV с инвойсами; система валидирует и импортирует записи, создавая при необходимости клиентов/проекты/категории.
- Пользователь просматривает список инвойсов с фильтрацией и пагинацией.
- На дашборде пользователь видит исторический график доходов и прогноз на N месяцев.
- Пользователь экспортирует свои инвойсы в CSV.

5. Внешние интеграции / Аналитика
- Аналитика реализована с помощью Pandas и scikit-learn (LinearRegression).
- Визуализация: Plotly.
- (Опционально) REST API: Django REST Framework для внешних подключений.

6. Оценка соответствия требованиям
- Минимум 3 кастомные модели — выполнено.
- Интеграция/аналитика — Pandas + scikit-learn + Plotly — выполнено.
- Импорт данных — реализован через веб-форму и management command — выполнено.
